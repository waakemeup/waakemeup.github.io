从零开始的React(1)

FrontEnd
  |  
 阅读次数

今天是我正式开始学习React的第一天,记录一下今天学到了什么

  1. 创建项目
    创建一个React项目一般使用CRA这种脚手架,具体为npx create-react-app ${项目名},当然现在有了vite,也可以用vite创建React项目,选择React即可。
    我在创建项目的时候遇到一个问题就是我在全局安装了webpack,和CRA要求的webpack版本不一致,这时在根目录创建一个.env文件即可,里面写入SKIP_PREFLIGHT_CHECK=true

  2. JSX
    JSX是JavaScript和HTML结合的一种写法,在其中class改为className,for改为htmlFor,其余都差不多,它的最外层只能被一个标签包裹,可以为<></>这样,这种写法被称为Fragments。
    像p,h1,a这类标签必须在中间写内容,否则会warning。
    还有就是props,我理解它是传递给组件的参数,可以对它采用ES6的解构赋值,还有就是它那啥只有一个{},而不是Vue中mustache那种必须{{}},这个比较值得注意的是CSS in JS,比如
    const headingStyle = { color:'red', backgroundColor:'black' }
    然后在JSX中style={headingStyle},像background-color,都会采用骆驼式命名法(Camel-Case),即它会变为backgroundColor。
    还有就是props可以有默认值,用${组件}.defaultProps = {title:'Task Tracker'},以及给props指定类型,此处有一个坑,需要先引入import PropsTypes from 'prop-types'然后比如${组件}.propTypes = {title:PropsTypes.string.isRequired}坑就是${组件}.后面propTypes第一个p是小写的,要注意(说简单吧也挺简单的

  3. 定义组件的两种方式
    编写JavaScript函数

    function Welcome(props) {
return <h1>Hello, {props.name}</h1>;
}

    采用ES6的class

    class Welcome extends React.Component {
  render() {
    return <h1>Hello, {this.props.name}</h1>;
  }
}

    这是复制粘贴官网的别骂了…

  4. Hook ? (因为不熟悉所以少说)
    今天接触的Hook有useLocation,useEffect,useState…但是,我还不是特别明白…
    Hook的意思是钩子,好像就是一些JavaScript函数(说错了不要喷我),以use开头,也可以有自定义的,然后这应该是我后面学习的重点,现在还完全没理解。

  5. 路由
    React中没有Vue中router-view这种东西,首先需要npm i react-router-dom,在组件中引入import {BrowserRouter as Router,Route} from 'react-router-dom'
    as就是取个别名,应该都懂吧,然后Router(值指B开头那个)需要包裹整个被路由的组件,也就是把放在最外层,然后里面有,比如</Route path='/about' component={About}>表示这个路由到/about这个页面,渲染的组件为About,若要渲染的东西是一堆JSX,可以写为<Route path='/' exact render={()=>(你的JSX代码)}/>,然后如果你在路由的组件中写了a标签,会重载页面,可以使用Link,import {Link} from 'react-router-dom',用法为<Link to='/'>Go back</Link>,这样页面不会重载。

    聊聊别的

    我发现我的VSCode写React没有JSX语法高亮,不是很好看,好烦啊
    我还去知乎提了个问 就是这个问题

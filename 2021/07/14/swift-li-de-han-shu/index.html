swift里的函数

Language
  |  
 阅读次数

Swift是一门什么语言?

简单来说,就是开源之敌Apple出品
能构建各种IOS应用
还有就是它是强类型的,和JavaScript啥的不一样

它的函数长这样

func mul(operand:Int,by:Int)->Int {
    return operand*by
}
mul(operand:5,by:6)

还可以加一些声明

func mul(_ operand:Int,by otherOperand:Int)->Int {
    return operand*otherOperand
}
mul(5,by:6)

_让传参时可以不写operand
